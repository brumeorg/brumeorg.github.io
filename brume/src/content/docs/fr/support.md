Besoin d'aide ? Trouvé un bug ? Envie de discuter d'un cas d'usage avant d'ouvrir une issue ? Voici comment nous joindre.

## Où aller

- **[Projet sur GitHub](/fr/support/github/)** — issues, discussions et code source.
- **[Parler à l'équipe](/fr/support/team/)** — contacts personnels des mainteneurs.

## Repères rapides

| Vous voulez… | Allez à |
|---|---|
| Signaler un bug | [Issues GitHub](https://github.com/brumeorg/Brume/issues) |
| Poser une question d'usage | [Discussions GitHub](https://github.com/brumeorg/Brume/discussions) |
| Suggérer une fonctionnalité | [Issues GitHub](https://github.com/brumeorg/Brume/issues) — vérifiez d'abord la [Roadmap](/fr/roadmap/) |
| Joindre un mainteneur directement | [Contacts de l'équipe](/fr/support/team/) |
| Parcourir la doc | [Documentation](/fr/docs/) |

Merci de ne pas ouvrir de rapport de bug privé pour des **problèmes de sécurité** — voir le [repo GitHub](https://github.com/brumeorg/Brume) pour la politique de divulgation responsable.
