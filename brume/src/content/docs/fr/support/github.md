Brume est développé en open source sur GitHub.

## Projet

- **Dépôt** — [github.com/brumeorg/Brume](https://github.com/brumeorg/Brume)

Le dépôt héberge le code source, les artefacts de release, le tracker d'issues et le forum de discussion.

## Issues

Utilisez les **issues** pour :

- Les rapports de bug — merci d'inclure votre version de Brume (`brume --help`), la version de Postgres et le `brume.yml` minimal qui reproduit.
- Les demandes de fonctionnalité — vérifiez la [Roadmap](/fr/roadmap/) d'abord pour voir si c'est déjà prévu.
- Les retours sur la documentation — fautes de frappe, sections obscures, pages manquantes.

→ [Ouvrir une issue](https://github.com/brumeorg/Brume/issues/new)

## Discussions

Utilisez les **discussions** pour :

- Les questions d'usage (« comment pseudonymiser ce type de schéma ? »)
- Les questions de conformité (« comment justifier ça devant mon DPO ? »)
- Les vitrines (« voici comment on a branché Brume dans notre CI »)

→ [Démarrer une discussion](https://github.com/brumeorg/Brume/discussions)

## Releases

Chaque release Brume est taguée et publiée sur la [page Releases](https://github.com/brumeorg/Brume/releases). Le changelog vit là.

## Sécurité

Merci de **ne pas ouvrir d'issues publiques pour des vulnérabilités de sécurité**. La politique de sécurité du repo décrit comment les signaler en privé.
