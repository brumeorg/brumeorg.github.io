Need help? Found a bug? Want to discuss a use case before opening an issue? Here's how to get in touch.

## Where to go

- **[Project on GitHub](/support/github/)** — issues, discussions, and source code.
- **[Talk to the team](/support/team/)** — personal contacts of the maintainers.

## Quick pointers

| You want to… | Go to |
|---|---|
| Report a bug | [GitHub issues](https://github.com/brumeorg/Brume/issues) |
| Ask a usage question | [GitHub discussions](https://github.com/brumeorg/Brume/discussions) |
| Suggest a feature | [GitHub issues](https://github.com/brumeorg/Brume/issues) — check the [roadmap](/roadmap/) first |
| Reach a maintainer directly | [Team contacts](/support/team/) |
| Browse the docs | [Documentation](/docs/) |

Please don't open private bug reports for **security issues** — see the [GitHub repo](https://github.com/brumeorg/Brume) for the responsible disclosure policy.
